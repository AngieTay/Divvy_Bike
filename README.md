# Divvy Bike
:bike: Divvy Bike Time Series Forecasting

## Time Series Forecasting Models Used:

1. sNaive - base model
2. sArima
3. Dynamic Harmonic Regression 
4. Vector Autoregressive Model (VAR)
5. Regression with Arima Errors
6. Neural Networks
